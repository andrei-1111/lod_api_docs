===============
Authorize Quote
===============

+---------------+--------------------------------------+
| **Resource:** | .. container:: notrans               |
|               |                                      |
|               |    /api/quote/<<quote id>>/authorize |
+---------------+--------------------------------------+
| **Method:**   | .. container:: notrans               |
|               |                                      |
|               |    POST                              |
+---------------+--------------------------------------+

This interface authorizes a quote.  Only quotes with a status of "Pending" can be authorized.
